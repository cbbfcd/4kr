## 变量和可变性

使用大型数据结构时，适当地使用可变变量，可能比复制和返回新分配的实例更快。

对于较小的数据结构，总是创建新实例，采用更偏向函数式的编程风格，可能会使代码更易理解，为可读性而牺牲性能或许是值得的

```rust
// 变量默认是 immutable 的
let x: u32 = 5;

// 可变变量 mut
let mut y: u32 = 5;

// 常量
// ⚠️变量虽然默认不可变，但是也不是等于就是常量了，语法上必须使用 const，不能有 mut，必须注明类型，总是不可变
const MAX: u32 = 10000;
```
